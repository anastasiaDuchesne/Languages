## Question words

- **Hur då?** = how
- **När** = when
- **Vilken** = which
- **då** is a placeholder for longer sentences in questions

## Prepositions

- **framåt** = forward
- **mellan** = between
- **bakom** = behind
- **hos honom** = at his place
- **före** = before
- **därför** = therefore
- **därför att** = because
- **eftersom** = because
- **Om** = if
- **men** = but
- **av en {something}** = of a
